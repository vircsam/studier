🚀 How to Run locally
Install Dependencies:

bash

cd /Users/meerthikasr/Desktop/studier
npm install
Run Dev Server:

bash

npm run dev
Open http://localhost:5173 in your browser.

Connecting Firebase: Create a .env file at the root of the project:

bash

cp .env.example .env
Replace the placeholders with your Firebase client details. The app will automatically connect and activate cloud database sync.

☁️ Deployment Instructions (Vercel)
Studier is structured to work out of the box on the Vercel Free Tier:

Push the code to a GitHub repository.
Link the repository to your Vercel account.
In Vercel Project Settings, add the environment variables listed in .env.example.
Vercel automatically deploys the frontend and hooks the /api directory serverless function handlers.
